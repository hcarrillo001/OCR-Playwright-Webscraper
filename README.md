# 🚀 Overview

OCR Playwright Webscraper is an AI-assisted browser automation project built in Python.

It expands on my previous GitHub project by integrating:

Natural language browser control

OCR (Optical Character Recognition)

Playwright MCP-based automation

Unlike traditional Playwright scripts, this project does not require writing browser automation code. Instead, you provide simple instructions in plain English, and the browser executes them accordingly.

# 👁️ OCR-Powered Data Extraction

One of the most powerful aspects of this project is how OCR dynamically reads on-screen content and structures it into usable data.

In testing, the OCR layer was able to:

Recognize betting odds directly from the rendered webpage

Interpret visual page elements instead of relying on HTML selectors

Extract the detected values

Automatically format the results into structured JSON

This demonstrates a shift from traditional DOM-based scraping to vision-based AI extraction, where automation interacts with the page the way a human visually would.
